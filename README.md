# 🏦 Banking System Implementation (C++)

A robust console-based Banking System application developed in C++. This project demonstrates core **Object-Oriented Programming (OOP)** principles and financial logic management.

## 🚀 Features
- **Account Management:** Create and manage customer accounts with unique IDs.
- **Secure Transactions:** Safe Deposit and Withdrawal operations.
- **Search Logic:** Implemented search functionality to find accounts by ID using efficient loops.
- **Data Encapsulation:** Used private access modifiers to protect sensitive balance and customer data.
- **User Interface:** Interactive menu-driven CLI for easy navigation.

## 🛠️ Technical Concepts Used
* **C++ Classes & Objects:** Building the blueprint for bank accounts.
* **Encapsulation:** Protecting data using Setters and Getters.
* **Control Flow:** Using `switch-case` and `for-loops` for transaction logic and account searching.
* **Input Validation:** Ensuring transactions are handled correctly.
